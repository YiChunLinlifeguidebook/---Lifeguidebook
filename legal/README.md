1. 至 [Meta AI — Llama License](https://ai.meta.com/llama/license/) 下載或複製**當前版本**的完整授權全文。
2. 將副本存成例如 `LLAMA_COMMUNITY_LICENSE.txt`（檔名可自訂，勿提交含商業機密之內容至公開 repo 若政策不允許）。
3. 產品介面請加上 **Built with Meta Llama** 標示（見 `badge-snippet.html`）。

若專案僅使用 LINE Bot 後端而未內嵌 Llama 推論，仍建議在對外說明或下載頁保留授權與標章，與實際使用範圍一致。
